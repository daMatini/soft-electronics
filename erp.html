견본과 양산

도금할때 견본과 양산 이란?
1.도금시 통상 Sample이란 1M2 이하작업시
예를 들어 2p/n, 5p/n 등을 통틀어 가격을 정한다 2. 도금 작업시 거래처에서 Sample Model 이라고 표시해도 1M2 이상 작업시
모델과 관계없이 양산 단가 적용한다.
3. 각공정시 Sample이라함은 소량작업시 작업조건을 컨트롤하여 진행하는 것으로 도금작업은 통상 1M2 이상은 모두 양산 단가 적용

도금 면적 환산지수 계산법

1M2 = 가로 1m x 세로 1m
1장(panel) 크기가 250mm x 350mm 경우 (250mm = 25cm = 0.25m)
0.25m x 0.35m = 0.0875m2 (1장 면적) (단위를 m으로 일치시켜 계산)
1 m2 크기는 몇장인가?
1/0.0875 = 11.42… 즉 11장 까지는 1 m2 보다 작다. (11장은 0.25x0.35x11=0.9625 m2 ) 12장은 1 m2 보다 크다. (12장은 0.25x0.35x12=1.05m2 )
이 경우 11장 까지는 견본으로 간주. 견본단가 12장 부터는 양산으로 간주. 양산 단가
